About ResourceFlow

The ResourceFlow API allows users to post, update, delete, and retrieve links to valuable content related to climate, technology, and more.

API Endpoints

  • GET /links: Retrieve all available links.
  • POST /links: Add a new link.
  • PUT /links/:id: Update a specific link.
  • DELETE /links/:id: Remove a link by its ID.

Overview

GET request: you can retrive all the links without an api key.

However, for POST, PUT & DELETE you must send your valid api key along with the request.

You can get your api key from here after creating an account

Getting All the Links


      // Example: GET request to retrieve all links
      fetch('http://localhost:4000/links', {
      method: 'GET',
      headers: {
      'content-Type': 'application/json'
      }
      })
      .then(response => response.json())
      .then(data => console.log(data))
      .catch(error => console.error('Error fetching links:', error));

Posting a Link


      // Example: POST request to post your link
      fetch('http://localhost:4000/links', {
      method: 'POST',
      headers: {
      'content-Type': 'application/json',
      'x-api-key': 'your-api-key-here'
      },
      body: "Your desired link(as a object) with the type and link(must) and optional sharedBy and description property" //make sure to JSON.stringify() it
      })
      .then(response => response.json())
      .then(data => console.log(data))
      .catch(error => console.error('Error fetching links:', error));

Deleting a Link(the link you have posted)


      // Example: DELETE request to delete your link
      fetch('http://localhost:4000/links/:id', {
      method: 'DELETE',
      headers: {
      'content-Type': 'application/json',
      'x-api-key': 'your-api-key-here'
      }
      })
      .then(response => response.json())
      .then(data => console.log(data))
      .catch(error => console.error('Error fetching links:', error));
